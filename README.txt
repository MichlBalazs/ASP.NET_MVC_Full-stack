HUN:
Ez egy full-stack, rétegzett, CRUD alkalmazás C#-ban írva, WPF grafikus interface-el, .NET 5-t használ, Visual Studio-ban lehet futtatni. Az alkalmazás egy fantasy boltnak az adatbázisát kezeli.

ENG:
This is a full-stack, layered CRUD application written in C# with a WPF graphical interface. It uses .NET 5 and can be run in Visual Studio. The application manages the database of a fantasy store.
